# AI Memory — экспорт памяти проекта b24

Это снимок персистентной памяти, накопленной при работе над проектом
(дашборд-надстройка над Bitrix24). Папка предназначена для переиспользования
контекста **другими нейросетевыми моделями** — просто дайте модели прочитать
эти файлы как фоновый контекст о проекте.

Источник: `~/.claude/projects/<...>/memory/`. Скопировано 2026-06-24.
Это **не живое состояние** — снимок на момент копирования. Файлы, функции и
флаги, упомянутые здесь, проверяйте по актуальному коду перед использованием.

## Как читать

- **`MEMORY.md`** — индекс: одна строка на каждый факт со ссылкой на файл.
  Начинайте отсюда, чтобы понять, что вообще есть.
- Остальные `*.md` — по одному факту/теме в файле. У каждого есть YAML-frontmatter:

  ```markdown
  ---
  name: <slug>
  description: <одна строка — о чём файл>
  metadata:
    type: user | feedback | project | reference
  ---
  <текст факта; ссылки на другие файлы вида [[name]]>
  ```

- `type`: `user` — кто пользователь; `feedback` — как с ним работать;
  `project` — рабочий контекст/цели/ограничения; `reference` — внешние ресурсы.
- `[[name]]` внутри текста — ссылка на другой файл памяти (по полю `name`).
